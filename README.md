# Biochem AI Learning Plan

This repository contains a student-centered **Biochemistry + Python + AI + Bioinformatics** learning plan built around small, portfolio-friendly projects.

The aim is not to turn the learner into a software engineer. The aim is to use Python and AI tools to support Biochemistry work such as:

- analyzing synthetic drug response data
- exploring protein sequences
- reading scientific abstracts more effectively
- interpreting simplified gene expression results
- turning project work into polished reports and portfolio artifacts

## Start Here

Read the documents in this order:

1. [Common Technical Setup and Workflow](docs/00_Common_Technical_Setup_and_Workflow.md)
2. Pick one project guide:
   [BioDose AI](docs/01_BioDose_AI_Drug_Response_Analyzer.md),

4. Use [Academic Life Science and Biochemistry Support Layer](docs/07_Academic_Life_Science_and_Biochemistry_Support_Layer.md) as a shared cross-project reference

## Suggested First Path

If you want the simplest path through the material:

1. Read [docs/00_Common_Technical_Setup_and_Workflow.md](docs/00_Common_Technical_Setup_and_Workflow.md)
2. Start with [docs/01_BioDose_AI_Drug_Response_Analyzer.md](docs/01_BioDose_AI_Drug_Response_Analyzer.md)
3. Build a small Gradio demo and one Quarto report
4. Expand later into ProteinLens or TargetReader AI

## Notes

- These documents are planning and guidance artifacts, not a runnable application by themselves.
- The guides intentionally emphasize synthetic or educational data first.
- AI-generated explanations are meant to support learning and scientific communication practice, not replace verification or biological judgment.
